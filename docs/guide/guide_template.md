# 你可以在这里写个自我介绍

`author` 会被作为链接的路径, 请限制为英文字符, 数字, `-` 和 `_`.

`author_nickname` 会被作为作者名放在文章开头的 note box 内.

`title` 会被显示在左侧的目录树上.

支持 Mathjax 渲染的 $\LaTeX$

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
